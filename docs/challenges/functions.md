# Funções

Essa pagina possui alguns desafios para ajudar no desenvolvimento do nosso jovem aprendiz.

## 🟢 Bloco 1 — Declaração e chamada básica

::: info Desafio 1
Crie uma função <code>saudar</code> que não recebe nada e exibe "Olá, mundo!" no console.
:::

::: info Desafio 2
Crie uma função <code>exibirNome</code> que recebe um nome e exibe "Meu nome é [nome]".
:::

::: info Desafio 3
Crie uma função <code>somar</code> que recebe dois números e exibe a soma no console.
:::

::: info Desafio 4
Crie uma função <code>apresentar</code> que recebe nome e idade e exibe uma frase completa.
:::

::: info Desafio 5
Crie uma função <code>exibirDia</code> que recebe um número (1 a 7) e exibe o dia da semana correspondente.
:::

::: info Desafio 6
Crie uma função <code>cumprimentar</code> que recebe nome e período do dia ("manhã", "tarde", "noite") e exibe a saudação correta.
:::

::: info Desafio 7
Crie uma função <code>contarAte</code> que recebe um número e exibe no console todos os números de 1 até ele.
:::

::: info Desafio 8
Crie uma função <code>repetirFrase</code> que recebe uma frase e um número, e exibe a frase aquela quantidade de vezes.
:::

## 🟡 Bloco 3 — Parâmetros opcionais e valores padrão

## 🟠 Bloco 5 — Funções que chamam outras funções

::: info Desafio 35
Crie <code>ehAprovado</code> que recebe uma nota e retorna <code>true</code>/<code>false</code>. Depois crie <code>exibirSituacao</code> que usa <code>ehAprovado</code> para exibir "Aprovado" ou "Reprovado".
:::

::: info Desafio 36
Crie <code>calcularAreaCirculo</code> usando uma função auxiliar <code>calcularQuadrado(n)</code> para elevar o raio ao quadrado.
:::

::: info Desafio 37
Crie <code>validarIdade</code> que retorna se é maior de idade. Crie <code>podeDirigir</code> que usa <code>validarIdade</code> para dar uma mensagem.
:::

::: info Desafio 38
Crie <code>normalizarNome</code> (tira espaços e coloca em minúsculo). Crie <code>compararNomes</code> que usa <code>normalizarNome</code> para comparar dois nomes e retornar se são iguais.
:::

::: info Desafio 39
Crie <code>calcularSubtotal(preco, qtd)</code> e <code>calcularTotal(subtotal, frete)</code>. Depois crie <code>finalizarPedido</code> que recebe preço, qtd e frete e usa as duas anteriores.
:::

::: info Desafio 40
Crie <code>ehVogal(letra)</code> que retorna <code>true</code>/<code>false</code>. Crie <code>contarVogais(frase)</code> que usa <code>ehVogal</code> para contar as vogais de uma string.
:::

::: info Desafio 41
Crie <code>celsiusParaFahrenheit(c)</code> e <code>fahrenheitParaCelsius(f)</code>. Depois crie <code>converterTemperatura(valor, unidade)</code> que chama a correta conforme a unidade passada.
:::

### 🔴 Bloco 6 — Arrow functions e callbacks introdutórios

::: info Desafio 42
Reescreva a função <code>dobrar</code> do desafio 9 usando arrow function.
:::

::: info Desafio 43
Reescreva a função <code>ehPar</code> do desafio 10 usando arrow function de uma linha.
:::

::: info Desafio 44
Crie uma arrow function <code>calcularImposto</code> que recebe um valor e retorna 15% dele.
:::

::: info Desafio 45
Use <code>array.map()</code> com uma arrow function para criar uma função <code>dobrarArray</code> que dobra todos os valores de um array.
:::

::: info Desafio 46
Use <code>array.filter()</code> com uma arrow function para criar <code>filtrarMaioresQue</code> que filtra números acima de um limite recebido.
:::

::: info Desafio 47
Crie uma função <code>aplicarOperacao(a, b, operacao)</code> que recebe dois números e uma função como terceiro parâmetro, e aplica essa função nos dois números.
:::

::: info Desafio 48
Use o desafio 47 para chamar <code>aplicarOperacao</code> passando uma arrow function de soma, depois uma de multiplicação.
:::

::: info Desafio 49
Crie uma função <code>executarDepois(fn, vezes)</code> que executa a função <code>fn</code> o número de vezes indicado.
:::

::: info Desafio 50
Crie uma função <code>pipeline(valor, ...funcoes)</code> que recebe um valor e uma lista de funções, aplicando cada uma em sequência no resultado anterior. Exemplo: <code>pipeline(5, dobrar, somarDez, ehPar)</code>.
:::
